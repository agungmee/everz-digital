Booking Procedures

  1. All booking via WhatsApp chat/call.
  2. Kami berikan informasi harga dan diskusi kebutuhan Anda.
  3. Setelah deal, kami berikan booking code (simpan dengan baik).
  4. Booking code berlaku hingga H-7 sebelum pick up date.
  5. Lakukan pembayaran atau DP sebelum booking code expired.
  6. Jika expired, booking tidak dapat kami jamin.
  7. Booking ulang diperlukan jika kode tidak valid.
  8. Admin akan kirim detail pembayaran & invoice via WhatsApp.
  9. Harga kami belum termasuk pajak pemerintah.

Refund Policy

  • H-7 sebelum pick up date: refund 50%.
  • Setelah H-7 pick up date: no refund.
  • Voucher policy tersedia (hubungi admin).
